### To install/upgrade yfinance  using pip, run:

#### $ pip install yfinance --upgrade --no-cache-dir

### To install streamlit using pip, run:

#### $ pip install streamlit

### To Start App, in terminal run:

#### $ streamlit run stock_prices.py
